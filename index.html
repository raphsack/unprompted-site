📦 How to Install

  1. Chrome/Edge:
    1. Download and extract the zip
    2. Open chrome://extensions
    3. Enable "Developer mode" (top right)
    4. Click "Load unpacked"
    5. Select the extracted folder
  2. Firefox:
    1. Download and extract the zip
    2. Open about:debugging#/runtime/this-firefox
    3. Click "Load Temporary Add-on"
    4. Select any file in the extracted folder

⚠️ Firefox: Currently requires manual installation as a temporary add-on (removed on browser restart). This is Firefox's security policy for unsigned extensions. We're working on store submission for permanent installation.
