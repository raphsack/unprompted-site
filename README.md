# Unprompted

**Know when to stop.** Free, privacy-first time tracking for AI tools.

🔗 **Live site:** https://raphsack.github.io/unprompted-site

## Features

- Track time on ChatGPT, Claude, Gemini, and other AI tools
- Set daily limits per site with gentle nudges
- Works in browser + Claude Code CLI
- Optional encrypted cross-device sync
- No accounts, no sign-up, data stays local

## Installation

**Browser Extension (Chrome/Firefox):**
- Download from the [website](https://raphsack.github.io/unprompted-site)
- Manual installation required (load as unpacked/temporary)

**Claude Code Plugin:**
```bash
npx unprompted install
```
